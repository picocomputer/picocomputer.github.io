1. Introduction¶

The Video Graphics Array is a Raspberry Pi Pico with RP6502-VGA firmware. Its primary connection is to a RP6502-RIA over a 5-wire PIX bus. More than one VGA module can be put on a PIX bus. Note that all VGA modules share the same 64K of VRAM and only one module can generate frame numbers and vsync interrupts.

2. VRAM and VREG¶

Virtual RAM (VRAM) is R/W memory that is accessed with the two VRAM portals provided by the RP6502-RIA. There is 64KB of shared VRAM.

Virtual registers (VREG) are set with an RP6502-API call. The standard VGA module is PIX ID 1.
