1. Introduction¶

The RIA runs a protected 32-bit kernel that you can call from the 6502.

2. Function Reference¶

The $ number in the headings are the API call identification numbers. See the RP6502-RIA documentation about calling the kernel.

$00 zvstack¶

void zvstack();

Abandon the vstack by resetting the vstack pointer. Not needed for normal operation, but some performance tricks can be achieved with this.

$01 open¶

int open(const char *path, int oflag);

Open a file.

Parameters

  • path – Filename

  • oflag – Flags

Retval -1

on error.

Retval >=0

The file handle number.
